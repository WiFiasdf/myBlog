# 前言

> **“望闻问切”** 是中医诊断疾病的基本方法，分别指 **观察、听嗅、询问、触诊** 四个步骤。

这套理论不仅适用于人体疾病的诊断，也可以灵活应用于电脑维修领域。将中医“四诊法”与电脑故障排查相结合，能够帮助我们更系统、更高效地识别并解决设备故障问题。

---

# 一、望：视觉观察，初步判断

在中医中，“望”指的是观察患者的面色、舌苔、精神状态等外在表现；而在电脑维修中，“望”则对应于对**硬件外观、接口状态、指示灯及系统界面**的观察，同时还需要对求助者加以留意。

## 1. 观察求助者

实际维修工作中，求助者情况千差万别，需要我们察言观色、见机行事：

- 若求助者神态**焦急、烦躁**，后续维修应加快节奏，交流时更加耐心礼貌。
- 若求助者态度**平和、不急**，可以从容细致地处理，把事情做好做全。

## 2. 观察需要维修的电脑

根据故障现象，对相应部位进行针对性观察：

| 故障现象 | 观察重点 |
| --- | --- |
| 蓝屏 | 蓝屏错误代码（Stop Code） |
| 发热严重、频繁卡顿 | 散热器积灰情况、运行程序数量、硬件性能是否老旧 |
| 无法开机 | 电源指示灯、主板灯、显示器信号 |

> 初学者不必望而却步，后续章节会对常见问题进行逐一讲解。

---

# 二、闻：听觉与嗅觉辅助判断

中医的“闻”包括听声音和嗅气味；在电脑维修中，“闻”主要体现在倾听部件异响，以及辨别是否有烧焦异味。

## 1. 异常噪音

- **风扇异响**：高速运转的噪音或摩擦声，可能意味着积灰严重或轴承磨损。
- **硬盘异响**：机械硬盘发出明显的“咔哒”声或“呲啦”声，往往是机械部件故障的信号。

## 2. 异常气味

- **烧焦味**：主板、电源或电容烧毁时常伴随刺鼻气味，这是电路短路或过载的重要警示。
- **霉味**：陈年老电脑散发霉味，多半是内部受潮长霉所致。

## 3. 异常提示音

部分主板配有蜂鸣器（Beep Speaker），开机自检若出现问题会发出报警音。不同的蜂鸣模式对应不同的故障类型，需查阅主板说明书对照排查。

# 三、问：了解使用情况，获取关键线索

中医通过“问”来了解病人的病史和生活习惯；在电脑维修中，“问”则是与用户沟通，了解设备使用环境、操作习惯、故障发生的时间与经过。

> 提问应有的放矢，围绕故障现象展开即可，不必事无巨细地逐项盘问。

> 同时也要意识到：病人可能会“撒点小谎”，客户也可能说不清楚毛病。因此需要通过追问与交叉验证，避免被片面信息误导。

## 重点询问方向

- **故障发生前的操作**：是否安装了新软件或新驱动？是否升级过系统？是否删除过重要文件？是否加装过新硬件？设备是否受过物理冲击？
- **使用频率与环境**：是否长时间高负载运行？是否处于高温或多尘环境？是否长期不关机？
- **历史维修记录**：此前是否出现过类似问题？是否更换过配件？上次维修是什么故障？

这些信息对于判断是**软件问题**还是**硬件损坏**至关重要，也能帮助我们有效缩小排查范围。

---

# 四、切：动手检测，精准定位

“切”在中医中是通过脉象判断病情；在电脑维修中，则是对设备进行实际测试与检测，包括使用工具软件、替换法、最小系统法等手段。

## 常用检测方法

| 方法 | 说明 |
| --- | --- |
| **硬件检测工具** | 使用 MemTest86 检测内存，CrystalDiskInfo 查看硬盘健康状态，DiskGenius 检查硬盘分区等 |
| **最小系统启动** | 仅保留主板、CPU、内存和显卡尝试开机，拔掉其余外设以排除干扰 |
| **替换法验证** | 用已知正常的电源、内存条、硬盘等逐一替换可疑部件，确认故障源 |
| **温度监测** | 使用 HWMonitor、AIDA64 等工具监控各组件温度，判断是否存在过热降频或关机 |

通过“切”，我们能够精准锁定问题根源，并为后续修复提供可靠依据。
